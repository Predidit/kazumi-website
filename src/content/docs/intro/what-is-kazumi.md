# Kazumi 是什么？

Kazumi 是使用 Flutter 开发的基于自定义规则的番剧采集与在线观看程序。使用最多五行基于 Xpath 语法的选择器构建自己的规则。
支持规则导入与规则分享，支持基于 Anime4K 的实时超分辨率，支持弹幕、一起看、投屏、硬件加速等超多功能。
